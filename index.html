🐾 Connect Your Google Sheet

Family members can edit the spreadsheet anytime — the guide updates automatically!

1
Make a copy of the template sheet
Go to File → Make a copy of the shared template, or create your own Google Sheet with the columns listed below.
2
Publish it to the web
In your sheet: File → Share → Publish to web → choose Sheet1 → format CSV → click Publish. Copy the URL it gives you.
3
Paste the URL below and hit Load!
📋 Required column names in your sheet (row 1 headers):
name emoji breed personality color meal1_time meal1_desc meal2_time meal2_desc meal3_time meal3_desc feeding_note supply1_icon supply1_label supply1_loc supply2_icon supply2_label supply2_loc supply3_icon supply3_label supply3_loc supply4_icon supply4_label supply4_loc med1_name med1_schedule med1_note med2_name med2_schedule med2_note emergency_name emergency_phone vet_name vet_phone
Each row = one cat. Leave cells blank if not applicable.
✅ Your current headers are supported: Name, Age, Color, MealTime, SpecialTreatment, FeedingLocation, Medicine

🏠 For the House Supplies section: add a second sheet tab named Supplies with columns: Category, Item, Location, Notes
Then publish that tab as CSV too and paste its URL below.
